# 灵境万象 · 疲劳岗位检测告警中心

基于 Vue3 + Vite 的疲劳驾驶检测系统前端展示 Dashboard

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

项目将自动在浏览器中打开 http://localhost:3000

### 3. 构建生产版本

```bash
npm run build
```

## 项目特性

### 技术栈
- Vue 3 (Composition API + script setup)
- Vite
- SCSS

### 核心功能

#### 1. 顶部导航栏
- 品牌 Logo 和标题
- 线路/班次/风险级别筛选器
- 实时时间显示
- AI 疲劳助手快捷入口

#### 2. 左侧面板
- **整体风险概况**：监控司机总数、高危/中等风险人数统计
- **司机风险排名**：可点击切换查看详情，按疲劳指数排序
- **设备在线状态**：穿戴设备和摄像头在线率

#### 3. 中间主面板
- **当前司机概要**：显示关注司机的基本信息和风险标签
- **智能穿戴体征**：心率、血氧、压力指数、体温实时监测
- **眼球检测**：
  - 疲劳指数仪表盘（0-100 动态渐变）
  - PERCLOS 闭眼比例
  - 视线偏移时长
  - 打哈欠频次
- **疲劳趋势图**：过去 30 分钟疲劳指数变化趋势

#### 4. 右侧面板
- **实时告警列表**：分级显示高危/预警/提示告警
- **班次风险分布**：早班/白班/夜班风险统计
- **AI 运营建议**：基于数据的智能决策建议

### 设计亮点

- **深色科技风**：深蓝-紫色渐变背景，青色/紫色/橙红高亮
- **响应式交互**：点击司机列表项实时切换详情面板
- **动态数据**：实时时钟、进度条、渐变仪表盘
- **视觉层次**：卡片阴影、发光边框、hover 动效
- **信息密度**：适合 1920x1080 大屏展示

## 数据说明

所有数据均为本地 mock，包括：
- 8 名虚拟司机的完整体征和行为数据
- 5 条不同级别的实时告警
- 3 个班次的风险分布统计
- 设备在线状态

## 截图建议

为获得最佳展示效果，建议：
1. 使用 1920x1080 分辨率浏览器窗口
2. 确保页面完全加载（时钟开始跳动）
3. 点击不同司机查看面板变化效果
4. 尝试使用顶部筛选器过滤数据
